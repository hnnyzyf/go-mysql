# MySql的协议声明

Protocol包主要实现的是mysql的的连接协议

## 介绍

Mysql的协议主要实现了如下两个过程:
    
- Connection Phase
- Command Phase

### 连接阶段

连接阶段 主要有两种方式

- Plain HandShake

![](image/plain_shake.png)

- SSL Handshake
